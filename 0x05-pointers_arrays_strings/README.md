# 0x05. C - Pointers, arrays and strings
## This repo contains code on the basics on pointers, arrays and strings in C Programminng
#### 0-reset_to_98.c - function that takes a pointer to an int as parameter and updates value to 98
#### 1-swap.c - fucntion that swaps two integer values, a, b to a=b b=a
#### 2-strlen.c - function that returns the length of a string (basic implementation of strlen in C)
#### 3-puts.c - function that prints a string to the stdout 
#### 4-print_rev.c - function that prints a given string in reverse
#### 5-rev_string.c - function that reverses a sting
#### 6-puts2.c - function that prints every other character of a string
#### 7-puts_half.c - function that prints half of a string
#### 8-print_array.c - function that prints n elements of an array of integers
#### 9-strcpy.c - function that copies the string pointed to by src to buffer pointed by dest
#### 100-atoi.c - function that convert a string to an integer
